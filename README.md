# QMAX2,3SAT-DHNN
QMAX2,3SAT is implemented into DHNN and Neuron states are updated for information fusion

The file "QMAX2,3SAT-DHNN_Asynchronous Update.py" is the main Python program, which implements logic-guided neuron updates.
The file "Tables and Proofs.pdf" contains supplementary tables that provide further explanations of the main text, as well as the proofs of all theorems presented in the paper.
The folder "Schematic Diagram" contains the source files of three schematic figures: Figure 1 (Graphical Abstract), Figure 2 (Schematic diagram of energy evolution), and Figure 3 (Schematic diagram of neuron updating).
The folder "Plotting Code" contains the source files of three schematic figures: Figure 1 (Graphical Abstract), Figure 2 (Schematic diagram of energy evolution), and Figure 3 (Schematic diagram of neuron updating).

The folder "Experimental Results" contains seven Excel files (.xlsx), where "Summary1.xlsx" provides a summary of the other six files organized by variable categories.
The folder "Plotting Code" contains seven Python scripts for generating the figures.
The folder "Figures of Experimental Results" contains the generated figures of the experimental results, produced using the provided plotting scripts.
